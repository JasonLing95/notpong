# notpong 🏓

A fast-paced, terminal-based Pong clone featuring character-perfect collisions and huge ASCII scores.

### Demo
https://github.com/user-attachments/assets/753b9464-eddd-4ad9-99f2-f7e3238f02b4



---

## 🚀 Quick Start

Play instantly with `uvx` (no install required):

```bash
uvx notpong
```

To install globally:

```bash
uv tool install notpong
```

---

## 🎮 Controls

- Move: Left/Right Arrows or A/D  
- Restart: R (on Game Over)  
- Quit: Q  

---

## 🖥️ Requirements

- Terminal window: 50x24 minimum  
- Python 3.8+
